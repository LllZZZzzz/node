- npm 包管理器 起因node 目前以扩展为一个生态圈

- fs文件系统 错误优先原则

    - 创建文件/文件夹/层级文件夹插件 mkdirp
    - 增删读 mkdir unlink readdir
- node snippets node提示插件
- supervisor 不用重启服务器
- silly-datetime 格式化日期
- node 模块化（一种规范）
